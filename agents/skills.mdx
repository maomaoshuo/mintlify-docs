Skills are modular capabilities that extend what a Dola Agent can do beyond conversation alone. Where a system prompt defines how your Agent thinks and speaks, skills define what it can *do* — retrieve live information from the web, generate images, connect to external services, process files, and more. You can add skills when creating an Agent, update them any time after publishing, and even sell your own custom skills independently on the marketplace so other creators can use them in their own Agents.

## What Is a Skill?

A skill is a self-contained capability that plugs into your Agent and activates when a buyer's request calls for it. The Agent decides when to invoke a skill based on the conversation — buyers don't need to trigger skills manually.

For example, an Agent with a web search skill can fetch current information in response to a question about today's news. An Agent with an image generation skill can produce a visual when a buyer describes what they want to see. Skills make your Agent dynamic and capable in ways that a system prompt alone cannot achieve.

<Note>
  Skills are composable — an Agent can have multiple skills active simultaneously. A single Agent could combine web search, image generation, and a custom API integration all at once, activating each as the conversation demands.
</Note>

## How to Add Skills to Your Agent

Adding skills to an Agent takes only a few clicks inside the Agent editor.

<Steps>
  <Step title="Open the Agent editor">
    From your Creator Dashboard on [dolaio.cn](https://dolaio.cn), open the Agent you want to update — or start a **New Agent** if you're configuring skills from scratch.
  </Step>
  <Step title="Navigate to the Skills tab">
    Inside the Agent editor, click the **Skills** tab. This is where you manage all capabilities attached to your Agent.
  </Step>
  <Step title="Click Add Skill">
    Click **Add Skill** to open the skill library. You'll see all available built-in skills organized by category, as well as any custom skills you've already created.
  </Step>
  <Step title="Choose a skill and configure it">
    Select the skill you want to add. Some skills work out of the box with no additional configuration. Others — particularly integration skills that connect to external services — may prompt you to provide credentials or settings specific to that service.
  </Step>
  <Step title="Save your changes">
    Click **Save**. The skill is now active on your Agent. If the Agent is already published, existing buyers will have access to the new skill immediately.
  </Step>
</Steps>

## Built-In Skill Categories

Dola's skill library is organized into four broad categories. Specific skills within each category may be added or updated over time — check the skill library in your dashboard for the current selection.

<CardGroup cols={2}>
  <Card title="Information Retrieval" icon="magnifying-glass">
    Skills in this category give your Agent the ability to find and surface information beyond its training data. This includes browsing the live web to answer current questions, reading and summarizing documents that buyers upload, and pulling in structured data from external sources.

    **Example uses:** research assistants, news summarizers, document Q&A tools, fact-checkers
  </Card>
  <Card title="Media Generation" icon="wand-magic-sparkles">
    Skills in this category let your Agent create original media content in response to buyer prompts. This includes generating images from text descriptions and producing audio output.

    **Example uses:** creative writing companions, brand asset generators, visual concept explorers, audio content tools
  </Card>
  <Card title="Data Processing" icon="table">
    Skills in this category handle structured information — performing calculations, transforming data between formats, processing uploaded files, and producing structured output that buyers can directly use or export.

    **Example uses:** spreadsheet assistants, report generators, data cleaning tools, invoice processors
  </Card>
  <Card title="Integration Skills" icon="plug">
    Skills in this category connect your Agent to external services and platforms. When a buyer interacts with your Agent, these skills can read from or write to third-party systems on their behalf — provided the appropriate credentials or permissions are configured.

    **Example uses:** CRM assistants, calendar schedulers, e-commerce helpers, notification tools
  </Card>
</CardGroup>

## Creating Custom Skills

If the built-in skill library doesn't cover your use case, you can build your own. Dola's skill configuration interface lets developers define custom skills that connect to any external API, run specific logic, or expose specialized capabilities that your Agent can invoke during a conversation.

Custom skills you create are private to your Agent by default. You can also choose to list them on the marketplace so other creators can add them to their own Agents — which opens a second revenue stream alongside your Agent sales.

## Selling Skills on the Marketplace

Skills are first-class products on Dola. You can list any skill — built-in or custom — independently on the marketplace, separate from any specific Agent. When another creator purchases your skill, they can add it to their own Agents just as they would a built-in skill.

This means you can build a business on Dola in two distinct ways:

<Tabs>
  <Tab title="Selling Agents">
    Build a fully configured Agent with a persona, instructions, model, and skills — then list it as a complete product. Buyers get a ready-to-use AI worker. You earn from every purchase or subscription.
  </Tab>
  <Tab title="Selling Skills">
    Build a valuable skill — a novel integration, a specialized data processor, a unique media generator — and list it standalone. Other creators add your skill to their Agents, and you earn each time it's purchased. Your skill can power dozens of Agents built by other people.
  </Tab>
</Tabs>

<Tip>
  Bundle complementary skills together in one Agent for a higher-value product. An Agent that can search the web, summarize documents, and format output for export is worth more — and sells more — than an Agent that only handles one of those tasks.
</Tip>

<Tip>
  List popular skills separately so other creators can enhance their Agents with your work. A skill that solves a common problem — like connecting to a widely used platform or handling a common file format — can generate significant passive income as the Dola creator ecosystem grows.
</Tip>
