Skill 是模块化能力，用来扩展 Dola Agent 在单纯对话之外能做的事情。系统提示词定义 Agent 如何思考和表达，Skill 则定义 Agent 可以实际执行什么动作，例如检索实时网页信息、生成图片、连接外部服务、处理文件等。你可以在创建 Agent 时添加 Skill，也可以在发布后随时更新，甚至可以把自定义 Skill 单独上架到市场，让其他创作者在自己的 Agent 中使用。

## 什么是 Skill？

Skill 是一种自包含能力，可以接入 Agent，并在买家的请求需要它时被调用。Agent 会根据对话判断何时调用 Skill，买家不需要手动触发。

例如，带有网页搜索 Skill 的 Agent 可以在回答今日新闻问题时获取最新信息。带有图像生成 Skill 的 Agent 可以在买家描述想看什么画面时生成视觉内容。Skill 让 Agent 变得动态且更有执行能力，这是单靠系统提示词无法实现的。

<Note>
  Skill 可以组合使用。一个 Agent 可以同时启用多个 Skill，例如网页搜索、图像生成和自定义 API 集成，并根据对话需要分别调用。
</Note>

## 如何为 Agent 添加 Skill

在 Agent 编辑器中，只需几次点击即可添加 Skill。

<Steps>
  <Step title="打开 Agent 编辑器">
    从 [dolaio.cn](https://dolaio.cn) 的 Creator Dashboard 中打开你要更新的 Agent；如果从零配置 Skill，则新建一个 **New Agent**。
  </Step>
  <Step title="进入 Skills 标签页">
    在 Agent 编辑器中点击 **Skills** 标签页。这里会管理所有附加到 Agent 的能力。
  </Step>
  <Step title="点击 Add Skill">
    点击 **Add Skill** 打开 Skill 库。你会看到按类别组织的内置 Skill，以及你已经创建过的自定义 Skill。
  </Step>
  <Step title="选择并配置 Skill">
    选择要添加的 Skill。有些 Skill 开箱即用，不需要额外配置。另一些 Skill，尤其是连接外部服务的集成类 Skill，可能会要求你提供对应服务的凭据或设置。
  </Step>
  <Step title="保存变更">
    点击 **Save**。该 Skill 会在 Agent 上生效。如果 Agent 已经发布，已有买家会立即获得新 Skill。
  </Step>
</Steps>

## 内置 Skill 类别

Dola 的 Skill 库按四个大类组织。每个类别下的具体 Skill 可能会随时间新增或更新，请在后台 Skill 库中查看当前可用列表。

<CardGroup cols={2}>
  <Card title="信息检索" icon="magnifying-glass">
    这类 Skill 让 Agent 能找到并呈现训练数据之外的信息，包括浏览实时网页回答当前问题、读取并总结买家上传的文档，以及从外部来源拉取结构化数据。

    **示例用途：** 研究助手、新闻总结器、文档问答工具、事实核查工具
  </Card>
  <Card title="媒体生成" icon="wand-magic-sparkles">
    这类 Skill 让 Agent 能根据买家提示生成原创媒体内容，包括根据文字描述生成图片，以及生成音频输出。

    **示例用途：** 创意写作伙伴、品牌素材生成器、视觉概念探索工具、音频内容工具
  </Card>
  <Card title="数据处理" icon="table">
    这类 Skill 处理结构化信息，例如计算、转换数据格式、处理上传文件，并生成买家可直接使用或导出的结构化输出。

    **示例用途：** 表格助手、报告生成器、数据清洗工具、发票处理工具
  </Card>
  <Card title="集成 Skill" icon="plug">
    这类 Skill 会把 Agent 连接到外部服务和平台。买家与 Agent 互动时，在配置了合适凭据或权限的前提下，这些 Skill 可以代表买家读取或写入第三方系统。

    **示例用途：** CRM 助手、日程安排工具、电商助手、通知工具
  </Card>
</CardGroup>

## 创建自定义 Skill

如果内置 Skill 库无法覆盖你的场景，你可以构建自己的 Skill。Dola 的 Skill 配置界面允许开发者定义自定义 Skill，用来连接任意外部 API、运行特定逻辑，或暴露 Agent 在对话中可以调用的专业能力。

你创建的自定义 Skill 默认只对你的 Agent 私有。你也可以选择把它上架到市场，让其他创作者把它添加到自己的 Agent 中。这会在 Agent 销售之外，为你打开第二条收入来源。

## 在市场销售 Skill

Skill 是 Dola 上的一等商品。你可以把任何 Skill，无论内置还是自定义，作为独立产品上架到市场，不依附于某个具体 Agent。其他创作者购买你的 Skill 后，可以像添加内置 Skill 一样把它加入自己的 Agent。

这意味着你可以用两种方式在 Dola 上建立业务：

<Tabs>
  <Tab title="销售 Agent">
    构建一个包含角色、指令、模型和 Skill 的完整 Agent，然后作为完整产品上架。买家获得的是即开即用的 AI 工作者。你从每次购买或订阅中获得收入。
  </Tab>
  <Tab title="销售 Skill">
    构建一个有价值的 Skill，例如新的集成、专业数据处理器或独特媒体生成器，并单独上架。其他创作者会把你的 Skill 添加到他们的 Agent 中，你会在每次购买时获得收入。一个 Skill 可以驱动许多由他人构建的 Agent。
  </Tab>
</Tabs>

<Tip>
  把互补 Skill 组合进同一个 Agent，可以形成更高价值的产品。一个能搜索网页、总结文档并格式化导出结果的 Agent，通常比只能完成单一任务的 Agent 更有价值，也更容易销售。
</Tip>

<Tip>
  把热门 Skill 单独上架，方便其他创作者用你的成果增强他们的 Agent。一个能解决常见问题的 Skill，例如连接常用平台或处理常见文件格式，会随着 Dola 创作者生态增长而持续产生被动收入。
</Tip>
