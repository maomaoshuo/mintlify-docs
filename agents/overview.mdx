Dola Agents are AI-powered digital workers you build, configure, and sell directly on the Dola marketplace. Each Agent combines a distinct persona, a set of instructions, a choice of AI model, and optional skills into a single product that buyers can purchase and start using immediately — no setup, no deployment, no waiting. Whether you're building a customer support specialist, a coding assistant, or a creative writing partner, Dola gives you the tools to bring that Agent to market and earn from it.

## What Is a Dola Agent?

A Dola Agent is more than a chatbot. It's a fully configured AI worker with its own identity and purpose. Every Agent on the platform is made up of four core components:

- **Persona** — The Agent's name, avatar, and public description. This is what buyers see when browsing the marketplace.
- **Instructions** — A system prompt that defines how the Agent behaves, what it knows, how it responds, and what tone it takes.
- **Model** — The underlying AI model that powers the Agent's reasoning and responses.
- **Skills** — Optional modular capabilities (such as web search, image generation, or API integrations) that extend what the Agent can do beyond conversation alone.

Together, these components define an Agent that is immediately useful to the people who buy it.

## The Agent Lifecycle

Every Agent on Dola moves through a clear lifecycle from idea to active use. Here's how that journey works:

<Steps>
  <Step title="Create">
    You open the Creator Dashboard on dolaio.cn and start a new Agent. You give it a name, upload an avatar, and write a public description that tells buyers what it does.
  </Step>
  <Step title="Configure">
    You write the system prompt that defines the Agent's persona and behavior, select an AI model, and optionally add skills to extend its capabilities.
  </Step>
  <Step title="Publish">
    You set a price — one-time or subscription — preview the Agent, test its responses, and click **Publish** to list it on the marketplace.
  </Step>
  <Step title="Sell">
    Buyers discover your Agent on the marketplace, read your description, and purchase it. Dola handles the payment automatically.
  </Step>
  <Step title="Buyer Activates Instantly">
    The moment a buyer completes their purchase, the Agent is live and ready to use. No deployment steps, no API keys to configure — it just works.
  </Step>
</Steps>

## What You Can Configure

Dola gives you granular control over every aspect of your Agent. The table below summarizes the key settings available in the Agent editor:

| Setting | What It Controls |
|---|---|
| **Name** | The Agent's display name across the marketplace and in the buyer's workspace |
| **Avatar** | The visual identity of your Agent — recommended square image, at least 256×256 px |
| **Description** | The public-facing summary buyers read before purchasing |
| **System Prompt** | The core instructions that govern behavior, tone, knowledge boundaries, and response style |
| **Model** | The AI model powering the Agent's reasoning and responses |
| **Skills** | Modular capabilities added on top of the base model |
| **Pricing** | One-time purchase price or recurring subscription |

You can update any of these settings after publishing. Changes to the model or instructions take effect immediately for all existing buyers.

## How Buyers Experience Your Agent

When a buyer finds your Agent on the marketplace, here's what they experience:

1. They read your public description and understand exactly what the Agent does.
2. They complete the purchase — Dola handles billing automatically.
3. The Agent appears instantly in their workspace, ready to use.
4. They start a conversation and the Agent responds according to your instructions and persona.

Buyers never deal with deployment, hosting, or model configuration. That complexity is entirely hidden. What they get is a working, purposeful AI assistant from the first message.

<Note>
  Dola covers initial traffic costs so buyers can try your Agent before committing fully. This lowers the barrier to purchase and increases your conversion rate.
</Note>
