Dola Agent 是你可以构建、配置并直接在 Dola 市场销售的 AI 数字工作者。每个 Agent 都把角色、指令、AI 模型和可选 Skill 组合成一个完整产品。买家购买后即可开始使用，无需设置、无需部署、无需等待。无论你想做客服专家、代码助手，还是创意写作伙伴，Dola 都能帮助你把 Agent 推向市场并产生收入。

## 什么是 Dola Agent？

Dola Agent 不只是聊天机器人。它是一个已经配置好的 AI 工作者，拥有自己的身份和目标。平台上的每个 Agent 都由四个核心部分组成：

- **Persona** — Agent 的名称、头像和公开描述。这是买家在市场中首先看到的内容。
- **Instructions** — 系统提示词，用来定义 Agent 的行为、知识边界、回复方式和语气。
- **Model** — 驱动 Agent 推理和回答的底层 AI 模型。
- **Skills** — 可选的模块化能力，例如网页搜索、图像生成或 API 集成，用来扩展 Agent 在对话之外能完成的动作。

这些组件共同定义了一个买家购买后马上能用的 Agent。

## Agent 生命周期

每个 Dola Agent 都会经历从想法到实际使用的清晰流程：

<Steps>
  <Step title="创建">
    你在 dolaio.cn 打开 Creator Dashboard，并新建一个 Agent。你需要为它命名，上传头像，并撰写公开描述，让买家知道它能做什么。
  </Step>
  <Step title="配置">
    你编写系统提示词，定义 Agent 的角色和行为；选择 AI 模型；并按需添加 Skill 来扩展能力。
  </Step>
  <Step title="发布">
    你设置价格，可以选择一次性购买或订阅；预览 Agent，测试回复效果，然后点击 **Publish** 上架到市场。
  </Step>
  <Step title="销售">
    买家在市场发现你的 Agent，阅读描述并完成购买。Dola 会自动处理支付。
  </Step>
  <Step title="买家立即激活">
    买家完成购买后，Agent 会立即可用。无需部署步骤，也无需配置 API Key。它会直接开始工作。
  </Step>
</Steps>

## 你可以配置什么

Dola 允许你细致控制 Agent 的各个方面。下表总结了 Agent 编辑器中的关键设置：

| 设置 | 控制内容 |
|---|---|
| **Name** | Agent 在市场和买家工作区中的显示名称 |
| **Avatar** | Agent 的视觉身份，建议使用至少 256×256 px 的正方形图片 |
| **Description** | 买家购买前会阅读的公开简介 |
| **System Prompt** | 控制行为、语气、知识边界和回复风格的核心指令 |
| **Model** | 驱动 Agent 推理和回答的 AI 模型 |
| **Skills** | 添加在基础模型之上的模块化能力 |
| **Pricing** | 一次性购买价格或周期订阅价格 |

发布后，你仍然可以更新这些设置。模型和指令变更会立即对所有已有买家生效。

## 买家如何体验你的 Agent

买家在市场中找到你的 Agent 后，会经历以下流程：

1. 阅读公开描述，明确 Agent 能做什么。
2. 完成购买，Dola 自动处理支付。
3. Agent 立即出现在买家工作区中，并可直接使用。
4. 买家开始对话，Agent 按你的指令和角色设定进行回复。

买家不会接触部署、托管或模型配置。这些复杂性都被平台隐藏起来。买家拿到的是一个从第一条消息起就能工作的 AI 助手。

<Note>
  Dola 会覆盖初始流量成本，让买家能在完全投入使用前先体验你的 Agent。这会降低购买门槛，并提升转化率。
</Note>
