Dola 智能体云端会员的核心价值，是给你一台可以持续工作的 **云端会员服务器**。所有 AI Agent 都可以在这台云端会员服务器里运行，不需要你电脑挂机，不需要自己搭建环境，也不需要为每个 Agent 单独准备服务器。

开通会员后，你可以免费使用 Dola 提供的主流、热门 Agent 服务。Dola 会把 Agent、模型、云端运行和入口连接整合在一起，让你直接在微信、飞书 / Lark 或 Dashboard 中调用这些 Agent。

会员套餐的核心价值是三件事：**云端会员服务器承载 Agent 工作**、**免费使用热门 Agent 服务**、**提供模型与智能体调用额度**。

## 选择建议

- **只想快速体验** — 选择 Trial 体验版，先看看云端会员服务器和热门 Agent 能做什么。
- **日常使用 AI Agent** — 选择 Basic 基础版，适合内容生成、资料整理、轻量办公和日常自动化。
- **持续业务使用** — 选择 Pro 专业版，适合更高频的业务任务、团队协作和持续运行场景。

  套餐价格、额度和可启动的智能体范围可能会随平台调整。购买前请以商品页展示的信息为准。

## 开通后怎么用

<Steps>
  <Step title="选择套餐">
    在 Agent 商店中打开 Dola 智能体云端会员商品，选择 Trial、Basic 或 Pro。
  </Step>
  <Step title="完成购买">
    确认套餐、金额和有效期后完成支付。
  </Step>
  <Step title="进入我的智能体">
    购买成功后，进入 **我的智能体** 查看云端会员服务器状态、免费用量、钱包余额和模型选择。
  </Step>
  <Step title="绑定使用入口">
    在实例页的 **入口链接** 中绑定微信、飞书 / Lark 或进入 Dashboard，开始使用 Agent。
  </Step>
</Steps>

<Tip>
  如果你还不确定要长期使用哪种 Agent，可以先用 Trial 体验版试用；确认场景后，再升级到 Basic 或 Pro。
</Tip>
