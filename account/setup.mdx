Dola 账号是你在 Dolaio.cn 中国独立服务中的统一身份。你可以用它进入 Agent 商店购买商品，查看自己的 Agent 实例，管理 AI 支付钱包，创建模型 API Key，并把 Agent 绑定到微信、飞书 / Lark 或 Dashboard 中使用。

国际服为 [dola.io](https://dola.io)。中国大陆地区用户请使用 [dolaio.cn](https://dolaio.cn)。

## 注册或登录

<Steps>
  <Step title="打开 Dola 服务">
    访问 [dolaio.cn](https://dolaio.cn)。如果你要直接浏览和购买商品，可以从 Dola 进入 Agent 商店。
  </Step>
  <Step title="点击登录或注册">
    当你点击 **我的智能体**、购买商品、进入 AI 支付、创建 API Key 或访问个人设置时，如果当前没有登录，页面会提示你登录或注册账号。
  </Step>
  <Step title="按页面提示完成登录">
    根据页面提供的登录方式完成账号验证。登录成功后，你可以继续刚才的购买、支付、模型或 Agent 管理操作。
  </Step>
  <Step title="确认账号状态">
    登录后，顶部或个人菜单会显示你的账号入口。后续购买、Agent 实例、钱包余额、API Key 和推荐返佣都会归属到这个账号下。
  </Step>
</Steps>

<Note>
  具体登录方式可能会随平台更新调整。请以当前页面展示的登录入口和验证方式为准。
</Note>

## 第一次使用建议

如果你是第一次使用 Dola，可以按下面顺序完成基础配置。

<Steps>
  <Step title="先进入 Agent 商店">
    打开 [shop.dolaio.cn/ai/store](https://shop.dolaio.cn/ai/store)，浏览 Dola 精选、会员套餐和不同分类的 Agent 商品。
  </Step>
  <Step title="选择一个 Agent 或会员套餐">
    如果只是体验，可以先选择适合新手的 Agent 或云端会员套餐。商品页会展示名称、介绍、价格、购买按钮和可分享链接。
  </Step>
  <Step title="完成购买">
    点击商品页中的 **购买**，在购买详情弹窗中确认数量、套餐和金额。支付成功后，系统会创建对应的 Agent 实例或开通对应权益。
  </Step>
  <Step title="进入我的智能体">
    打开 **我的智能体**，找到刚购买的 Agent。实例页会展示运行状态、入口链接、免费用量、钱包余额和模型选择。
  </Step>
  <Step title="绑定日常入口">
    在实例页的 **入口链接** 区域绑定微信、飞书 / Lark 或进入 Dashboard。绑定后，你就可以在最熟悉的沟通工具中使用 Agent。
  </Step>
</Steps>

## 钱包和模型设置

购买和使用 Agent 后，你可能还需要管理 AI 支付和模型。

- **AI 支付** — 前往 [shop.dolaio.cn/ai](https://shop.dolaio.cn/ai)，查看钱包余额、充值入口、模型 Token 费用和 Agent 支付记录。
- **模型服务** — 前往 [shop.dolaio.cn/model](https://shop.dolaio.cn/model)，查看模型列表、价格、API Key、接入说明和模型检测。
- **Agent 模型选择** — 在 Agent 实例页中查看当前模型，并在有权限时切换到其他可选模型。
- **智能体钱包** — 如果你希望 Agent 在授权范围内完成购物、支付或转账，可以在 AI 支付中配置智能体钱包。

## 如果你想销售或代理

Dola 账号也可以用于参与销售、推荐和代理合作。

- **分享商品返佣** — 在具体商品页复制分享链接。不同商品的返佣比例可能不同，佣金按商品规则计算。
- **成为代理商** — 如果你想拥有自己的 Agent 商店并推广平台产品，可以查看 [成为代理商](/marketplace/resellers)。
- **发布商品** — 如果你要上架 AI Agent、结果型 Agent、AI 硬件产品或个人服务，可以先查看 [Agent 商店总览](/marketplace/overview) 了解支持的商品类型。
