Dola 账号设置让你控制自己在平台上的展示方式、接收哪些活动通知，以及如何保护账号安全。点击右上角头像，并从下拉菜单选择 **Account Settings**，即可访问所有设置。

---

## 个人资料设置

你的个人资料是你在 Dola 上的公开身份。买家浏览市场时，可以访问你的创作者页面，了解你并查看你发布的 Agent。因此，保持资料准确、完整，会直接影响你的创作者可信度。

- **Display name** — 更新显示在商品、评价和平台各处的名称
- **Avatar** — 上传或替换头像；至少 256×256 px 的正方形图片效果最好
- **Bio** — 编辑简短介绍，说明你的专业背景、经验，或擅长构建哪类 Agent
- **Public profile URL** — 你的唯一创作者主页链接，例如 `dolaio.cn/@yourname`，所有查看你商品的买家都可以访问
- **Social links** — 可选添加 GitHub、X（Twitter）、LinkedIn 或个人网站链接；这些链接会显示在创作者资料中，帮助买家进一步了解你

要更新这些信息，请前往 **Account Settings → Profile** 并保存变更。

## 通知偏好

Dola 会通过邮件和站内通知，让你了解与自己相关的重要活动。你可以精确选择哪些事件触发通知，以及通过哪个渠道接收。

**Email notifications** — Dola 可以向你发送邮件提醒：
- 你的 Agent 产生了新销售
- 商品收到新评价
- 提现已成功处理
- 市场公告和平台更新

**In-app notifications** — 实时显示在 Dola 面板中：
- 收到买家消息
- 提交审核或近期发布的 Agent 出现状态更新

要管理偏好，请前往 **Account Settings → Notifications**，并逐项开启或关闭通知类型。变更会立即生效。

## 支付与收款设置

所有账单和收款活动都在 **Account Settings → Payments** 中管理。

**对创作者：**
- 添加或更新收款方式，确保收益发送到正确账户
- 查看即将到账和历史提现，包括金额和处理日期
- 如需完整了解提现周期、费用和最低门槛，请查看 [结算](/payments/settlements-old)

**对买家：**
- 查看完整账单历史，包括过去的购买和发票
- 管理活跃 Agent 订阅；你可以随时在这里取消或修改订阅
- 更新未来购买使用的支付方式

<Note>
  如果你同时以买家和创作者身份使用 Dola，账单历史和收款信息都会显示在同一个 Payments 区域中，并按不同标签组织。
</Note>

## 安全

如果你有正在销售的商品和收益，保护 Dola 账号尤其重要。Dola 提供了多种工具帮助你保持账号安全。

**修改密码**

前往 **Account Settings → Security → Change Password**。输入当前密码，再输入两次新密码确认。建议选择至少 12 个字符、且只用于 Dola 的唯一密码。

**启用双因素认证（2FA）**

双因素认证会为账号增加第二层保护，强烈建议所有创作者账号启用。

设置方式：
1. 前往 **Account Settings → Security → Enable 2FA**
2. 使用 Google Authenticator 或 Authy 等认证器应用扫描屏幕上的二维码
3. 输入应用中显示的六位验证码完成确认
4. 将备份代码保存在安全位置；如果以后无法访问认证器应用，你会需要这些代码

**查看活跃会话**

在 **Account Settings → Security → Active Sessions** 下，你可以查看当前登录账号的所有设备和位置。如果发现不熟悉的会话，点击旁边的 **Sign Out** 即可立即撤销访问。

## 删除账号

如果你想永久关闭 Dola 账号，请前往 **Account Settings → Danger Zone → Delete Account**，并按照确认步骤操作。

<Warning>
  删除账号不可逆。你的所有活跃商品都会从市场移除，个人资料将不再可访问，任何待结算收益会在账号关闭前按照 Dola 服务条款处理。发起删除前，请先提现所有可用余额。
</Warning>
