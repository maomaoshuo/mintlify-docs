## 通用问题

<AccordionGroup>

  <Accordion title="Dola 是什么？">
    Dola 是一个 Agent 商店平台。创作者可以发布 24 小时独立运行的 AI Agent、直接调用即可获得结果的结果型 Agent、AI 硬件产品和个人服务；买家可以在市场中发现、购买并使用这些商品。平台会处理发现、支付和交付，让创作者专注产品，让买家专注获得结果。
  </Accordion>

  <Accordion title="Dola 适合谁？必须是开发者才能用吗？">
    Dola 面向广泛用户。买家不需要技术背景，只需浏览市场、购买符合需求的 Agent，并通过几次点击完成激活。创作者如果熟悉 AI 工具或工作流会更有优势，但 Dola 也支持无代码方式，让非开发者也能创建 Agent。需要接入自定义逻辑或 API 的开发者，可以通过 Agent 编辑器中的技术配置选项完成。
  </Accordion>

  <Accordion title="Dola 在我的国家或地区可用吗？">
    Dola 面向全球用户开放 Agent 浏览和购买。创作者收款能力可能会因地区和支持的支付服务商而异。账号设置过程中，平台会提示你所在位置可用的收款方式。如果你对某个国家或地区的可用性有疑问，请联系 Dola 支持团队。
  </Accordion>

</AccordionGroup>

## 购买 Agent

<AccordionGroup>

  <Accordion title="购买后如何激活 Agent？">
    完成购买后，前往 **Dashboard → My Agents**，找到刚购买的 Agent。点击 **Activate**，并按照创作者提供的任何设置步骤操作。这些步骤可能包括连接外部服务或输入配置值。激活后，Agent 会立即可用。
  </Accordion>

  <Accordion title="购买前可以试用 Agent 吗？">
    部分创作者会为 Agent 提供免费试用层级或有限演示版本。你可以在商品页寻找 **Try for Free** 或 **Demo** 按钮。如果没有试用，请仔细查看商品描述。许多创作者会提供截图、视频或详细使用场景说明，帮助你在购买前做出判断。
  </Accordion>

  <Accordion title="退款政策是什么？">
    Dola 会为符合条件的购买提供退款窗口。具体条件和时限请查看商品页以及 Dola 的退款政策文档。退款请求会逐案评估；已经大量使用或不存在缺陷的 Agent 可能不符合退款条件。如需申请退款，请前往 **Dashboard → My Agents → [Agent name] → Request Refund**，并说明原因。
  </Accordion>

  <Accordion title="如果创作者更新了我购买的 Agent，会发生什么？">
    你已经购买的 Agent 获得更新后，通常会自动向你开放，且不额外收费。创作者发布更新时，你会在面板中看到通知。有些更新会即时生效，有些可能要求你重新激活或重新配置 Agent。创作者的更新说明会解释变更内容以及是否需要你操作。
  </Accordion>

</AccordionGroup>

## 创建和销售

<AccordionGroup>

  <Accordion title="创建 Agent 需要会写代码吗？">
    不一定。Dola 同时支持基于代码和无代码的 Agent 创建方式。如果你熟悉提示词工程、工作流构建器或可视化自动化平台，就可以不写代码创建并发布 Agent。想接入自定义逻辑或 API 的开发者，也可以通过 Agent 编辑器中支持的技术配置选项实现。
  </Accordion>

  <Accordion title="商品审核需要多久？">
    大多数 Agent 商品会在一到三个工作日内完成审核。提交高峰期可能略久。审核完成后，你会收到站内通知和邮件，了解商品是已通过、需要修改，还是被拒绝并附带反馈。为避免延迟，请在提交前确认商品描述、价格和分类完整且准确。
  </Accordion>

  <Accordion title="如何提高销量？">
    在 Dola 上提高销量的有效方式包括：撰写清晰详细的商品描述，加入演示视频或截图，及时回复买家消息，并主动收集满意客户评价。持续更新 Agent、快速修复反馈问题，也会长期提升声誉。资料完整且发布多个 Agent 的创作者，通常在市场搜索结果中排名更高。
  </Accordion>

  <Accordion title="Dola 市场现在支持哪些商品类型？">
    当前 Dola 市场支持四类商品：运行在 24 小时服务器中的 AI Agent、直接调用即可获得结果的结果型 Agent、AI 硬件产品，以及创作者提供的个人服务。Skill 主要作为 Agent 的内部能力使用，目前不作为独立商品销售。
  </Accordion>

</AccordionGroup>

## 支付

<AccordionGroup>

  <Accordion title="我什么时候能收到收益？">
    销售收益通常会在一段短暂清算期后滚动变为可提现余额。在清算期内，Dola 会处理交易并覆盖适用的退款窗口。资金清算完成后，你可以从 **Account Settings → Payments → Withdraw** 发起提现。完整提现周期请查看 [结算](/payments/settlements-old)。
  </Accordion>

  <Accordion title="提现有最低金额要求吗？">
    有。Dola 要求余额达到最低门槛后才能发起提现。具体门槛可能会根据你的地区和所选收款方式变化。当前余额和最低所需金额都会显示在 **Account Settings → Payments** 中，方便你随时确认状态。
  </Accordion>

  <Accordion title="买家可以使用哪些支付方式？">
    Dola 接受主流信用卡和借记卡，也支持部分数字支付方式。可用选项会在结账页展示，并可能根据你所在位置不同而变化。所有支付都会被安全处理；Dola 不会在服务器上保存完整卡号信息。
  </Accordion>

</AccordionGroup>

## 故障排查

<AccordionGroup>

  <Accordion title="我的 Agent 没有响应，该怎么办？">
    首先检查 **Dashboard → My Agents** 中的 Agent 状态指示。如果显示错误，请尝试停用并重新激活 Agent。同时确认 Agent 依赖的外部连接或 API Key 仍然有效且未过期。如果问题仍然存在，请查看创作者商品页是否有已知问题说明，或通过消息功能直接联系创作者。如果 Agent 长时间无法正常工作，你也可以向 Dola 支持团队报告。
  </Accordion>

  <Accordion title="没有收到验证邮件，如何重新发送？">
    如果几分钟后邮箱中仍没有验证邮件，请先检查垃圾邮件或广告邮件文件夹，自动邮件有时会被过滤。如果仍然没有，请回到 Dola 登录页面并点击 **Resend Verification Email**，系统会很快发送新的链接。请确认注册时使用的邮箱地址正确。如果仍遇到问题，请联系 Dola 支持团队，他们可以手动协助验证账号。
  </Accordion>

</AccordionGroup>
