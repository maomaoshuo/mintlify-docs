## 基础问题

<AccordionGroup>
  <Accordion title="Dola 是什么？">
    Dola 是一个 Agent 商店平台。用户可以购买和使用 AI Agent、结果型 Agent、AI 硬件产品和个人服务；创作者和代理商也可以围绕 Agent 商店进行销售、推荐和经营。
  </Accordion>

  <Accordion title="Dola 在中国大陆可以用吗？">
    国际服为 [dola.io](https://dola.io)，不向中国大陆地区提供服务。中国大陆地区用户请使用 [dolaio.cn](https://dolaio.cn)。
  </Accordion>

  <Accordion title="Dola 账号有什么用？">
    Dola 账号用于购买 Agent、进入我的智能体、管理 AI 支付钱包、创建模型 API Key、绑定微信或飞书入口，以及参与商品分享返佣。
  </Accordion>
</AccordionGroup>

## 使用 Agent

<AccordionGroup>
  <Accordion title="购买后在哪里使用 Agent？">
    购买或领取 Agent 后，你会进入 Agent 实例页。实例页会展示运行状态、入口链接、免费用量、钱包余额和模型选择。你可以从这里绑定微信、飞书 / Lark，或进入 Dashboard 使用和管理 Agent。
  </Accordion>

  <Accordion title="Agent 可以接入哪些入口？">
    当前文档重点说明了微信、飞书 / Lark 和 Dashboard。微信适合个人和移动端使用；飞书 / Lark 适合团队协作；Dashboard 更适合查看状态、用量、模型和管理信息。
  </Accordion>

  <Accordion title="会员套餐有什么用？">
    Dola 智能体云端会员提供云端会员服务器。所有 AI Agent 都可以在会员服务器里工作，不需要你电脑挂机或自己搭建环境。开通会员后，可以免费使用 Dola 提供的主流、热门 Agent 服务。
  </Accordion>
</AccordionGroup>

## AI 支付和模型

<AccordionGroup>
  <Accordion title="AI 支付页是做什么的？">
    AI 支付页用于管理钱包余额、充值入口、模型 Token 费用、Agent 支付记录和智能体钱包。它适合查看钱花在哪里，以及哪些 Agent 或模型调用产生了消费。
  </Accordion>

  <Accordion title="模型页是做什么的？">
    模型页用于查看可用模型与价格，管理 API Key，阅读接入说明，并测试模型是否可用。你可以在 Dola Agent 中选择模型，也可以在自己的私有场景中通过 API Key 接入模型服务。
  </Accordion>

  <Accordion title="API Key 要注意什么？">
    API Key 等同于模型调用权限。不要把它写进公开仓库、截图、文档示例或前端代码中。建议按项目或使用场景分别创建和管理。
  </Accordion>
</AccordionGroup>

## 推荐返佣

<AccordionGroup>
  <Accordion title="推荐返佣怎么产生？">
    在商品页分享时，系统会生成个人商品码。个人商品码包含原商品码和你的推荐码。买家通过个人商品码完成购买后，会按该商品的返佣规则计算佣金。
  </Accordion>

  <Accordion title="用户跳到其他商品还会保留代理关系吗？">
    会。用户通过个人商品码进入商店后，即使跳转到其他商品，系统也会保留代理关系。用户购买成功后，会正式确定代理商关系。
  </Accordion>

  <Accordion title="个人商品码和绑定代理商哪个优先？">
    个人商品码优先。如果用户通过某个个人商品码购买，该订单优先归属于这个个人商品码，并按个人商品码分账。只有没有新的个人商品码参与时，才按已绑定代理商关系归属。
  </Accordion>
</AccordionGroup>
