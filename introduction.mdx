Dola 是一个 AI Agent 市场。你可以在这里创建 Agent、购买 Agent，并把 Agent 直接投入工作。无论你是想把专业经验打包成 Agent 并出售的创作者，还是想直接使用成熟 AI Agent 的买家，Dola 都能把从创建到交付的流程放在同一个平台里完成。

## Dola 适合谁

**创作者** 可以在 Dola 市场设计并发布 AI Agent。你负责想法、角色和指令，Dola 负责模型接入、支付和交付。如果你有值得产品化的知识、流程或使用场景，就可以把它做成能持续产生收入的 Agent。

**买家** 可以在市场中浏览并购买即开即用的 Agent。完成购买后，Agent 会立即激活并可用。你不需要部署服务，不需要管理 API Key，也不需要处理复杂集成。打开后就能开始工作。

## Dola 有什么不同

### 模型选择自由
你可以从 Dola 精选的高质量模型中选择最适合 Agent 的模型。Dola 还为每个 Agent 覆盖初始流量成本，让买家能先体验再决定是否长期使用，从而降低首次成交门槛。

### 不只销售 Agent
Dola 市场不只支持 AI Agent。创作者也可以出售个人服务、自定义配置和独立 Skill。只要能为别人的工作流创造价值，就可以在 Dola 上产品化。

### 内置推荐返佣
每个用户都有专属推荐链接。分享链接后，只要有人通过你的链接完成购买，你就能获得佣金。推荐系统直接内置在平台中，不需要第三方工具，也不需要手动追踪。

### 端到端商业化
Dola 覆盖完整生命周期：创建 Agent，上架市场，交付给买家，再完成创作者收益结算。你不需要把创建工具、销售渠道和收款系统拼在一起。
