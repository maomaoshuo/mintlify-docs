Dola 内置支付系统，管理完整商业闭环：从买家完成结账，到创作者收到收益。你不需要接入第三方支付处理方，也不需要自己搭建账单基础设施。Dola 会自动完成收款、平台费扣除和结算，让你专注于构建或使用优秀 Agent。

## Dola 支付如何运作

Dola 上的每笔交易都遵循同一条流程：买家为 Agent、结果型 Agent、AI 硬件产品或个人服务付款，Dola 处理支付，扣除平台费用，并把剩余收入计入创作者余额。这个闭环让市场两侧都能获得稳定、可预期的体验。

## 对买家

购买商品时，你会通过 Dola 结账流程使用支持的支付方式直接付款。购买成功后，你会按商品类型获得 Agent 访问权限、结果型服务调用入口、硬件交付信息或个人服务预约说明。

- **使用支持的方式付款** — 信用卡和借记卡在所有地区可用；其他本地支付方式会根据你所在地区在结账页显示。
- **选择购买类型** — 商品可以是一次性购买（永久访问），也可以是订阅（月付或年付）。商品页会在你确认前清楚展示适用模式。
- **管理订阅** — 打开 **Billing Dashboard** 查看活跃订阅、更新支付方式或随时取消。

## 对创作者

Dola 会自动追踪每笔销售，并在 **Creator Dashboard** 中实时展示收益。

- **从每笔销售获得收入** — 买家每次购买你的商品，销售收入扣除平台费后会计入创作者余额。
- **一眼查看收益** — Creator Dashboard 会按商品、日期和交易类型展示完整收入明细。
- **按自己的节奏提现** — 余额可用后，你可以随时提现到已连接的收款方式。

## 平台费用

Dola 会从每笔成功销售中收取一定比例的平台费用。准确费率会在发布前显示在 **Creator Dashboard → Listings** 中，因此你可以在完全了解净收入的前提下定价。费用会自动扣除，你不需要手动计算或另行支付。

## 会员套餐初始额度

为了帮助新发布的 Agent 获得早期体验，Dola 会员套餐会为 Agent 提供初始使用额度。这意味着买家可以先开始使用新 Agent，帮助商品更有机会获得积极早期评价。

## 购买模式

| 模式 | 运作方式 |
|---|---|
| **一次性购买** | 买家支付一次，并获得对应商品的访问权限、调用次数、实物交付或服务权益。 |
| **订阅** | 买家按月或按年支付持续访问费用。 |

创建商品时，你可以选择最符合商品价值主张的模式。
