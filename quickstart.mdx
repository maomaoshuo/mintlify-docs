title: "Dola 快速开始：购买或发布你的第一个 AI Agent"
sidebarTitle: "快速开始"
description: "创建 Dola 账号，从市场购买现成 AI Agent，或配置并发布你自己的 Agent。"
---

Dola 可以让你在几分钟内从零开始使用 AI Agent。你可以选择购买一个现成 Agent，也可以进入创作者后台，创建并发布自己的 Agent。

<Note>
  Dola 会为每个 Agent 覆盖初始流量成本。买家激活后即可真实使用，创作者也可以让买家先体验 Agent 的效果，再决定是否购买。双方都不需要承担前置风险。
</Note>

<Tabs>
  <Tab title="购买 Agent">
    按照下面的步骤找到合适的 Agent，完成购买，并立即投入使用。

    <Steps>
      <Step title="创建 Dola 账号">
        前往 [dolaio.cn](https://dolaio.cn) 注册免费账号。你只需要一个邮箱即可开始。确认邮箱后，你会直接进入市场。
      </Step>
      <Step title="浏览市场">
        你可以按分类、使用场景或关键词浏览 Agent。使用左侧筛选器按能力、模型或价格缩小范围。每个商品页都会展示 Agent 的角色、能力和附带的 Skill。
      </Step>
      <Step title="购买 Agent">
        打开 Agent 商品页，查看完整描述、包含的 Skill 和价格。确认后点击 **Buy Now** 并完成结账。Dola 会处理支付流程，你无需配置任何内容。
      </Step>
      <Step title="连接并激活">
        购买完成后，Agent 会出现在你的 **My Agents** 面板中。点击 **Activate** 后，Agent 会立即可用。你可以马上开始第一段对话或任务，无需 API Key、部署步骤或等待。
      </Step>
      <Step title="按工作流自定义（可选）">
        激活后，你可以调整 Agent 设置，修改指令以匹配你的使用习惯，也可以从市场添加额外 Skill。自定义始终是可选项；Agent 会按照创作者配置好的方式开箱即用。
      </Step>
    </Steps>

    <Tip>
      不确定该选哪个 Agent？可以按 **Top Rated** 或 **Most Popular** 排序，查看其他买家最常使用的产品。商品页通常包含示例对话，方便你在购买前预览效果。
    </Tip>
  </Tab>

  <Tab title="创建并销售 Agent">
    按照下面的步骤创建 Agent，发布到市场，并开始获得收入。

    <Steps>
      <Step title="创建 Dola 账号">
        前往 [dolaio.cn](https://dolaio.cn) 注册。所有注册用户都可以使用创作者功能，不需要单独申请。
      </Step>
      <Step title="打开创作者后台">
        从账号菜单进入 **Creator Dashboard**，然后点击 **New Agent**。你会进入 Agent 创建器，并在这里完成从角色到定价的全部配置。
      </Step>
      <Step title="配置 Agent">
        填写基础信息：
        - **Name** — 给 Agent 一个清晰、好记的名字。
        - **Description** — 说明 Agent 能做什么、适合谁使用。这段内容会出现在市场商品页。
        - **Persona** — 定义 Agent 的语气、风格和角色，让每次互动保持一致。
        - **Instructions** — 编写系统级指令，决定 Agent 如何回答和行动。

        这里的配置越清晰，买家满意度和评分通常越高。
      </Step>
      <Step title="选择 AI 模型">
        从 Dola 支持的模型列表中选择驱动 Agent 的底层 AI 模型。不同模型擅长不同任务，请选择最适合 Agent 目标的模型。Dola 会覆盖初始流量成本，因此买家购买后可以立即开始使用，而不会把额外成本转嫁给你。
      </Step>
      <Step title="添加 Skill（可选）">
        Skill 是可以附加到 Agent 上的模块化能力，例如网页搜索、图像生成或自定义 API 调用。浏览 Skill 库并添加能提升 Agent 价值的能力。你也可以把 Skill 单独放到市场销售。
      </Step>
      <Step title="设置价格并发布">
        输入商品价格，预览市场页面，然后点击 **Publish**。Agent 会立即上架到 Dola 市场。每一笔销售的交付和支付处理都会由 Dola 自动完成。
      </Step>
      <Step title="分享推荐链接">
        发布后，从 **Creator Dashboard** 获取你的专属推荐链接。你可以把它分享到社交媒体、newsletter 或社群中。通过你的链接产生的每一笔购买，都会在 Agent 销售收入之外为你带来额外佣金。
      </Step>
    </Steps>

    <Tip>
      Agent 描述要写给买家看，而不是写给自己看。优先讲结果：买家能完成什么？围绕具体使用场景写的商品页，通常比泛泛介绍表现更好。
    </Tip>
  </Tab>
</Tabs>
