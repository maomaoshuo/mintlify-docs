Dolaio.cn 正在招募智能体代理合作方。代理商可以基于 Dola 的平台能力，拥有自己的 Agent 商店，向个人用户或企业客户推广、销售和交付智能体服务。

Dolaio.cn 不是单纯的 Token 转售平台。Token 聚合是基础能力，它让用户获得更灵活、更稳定的 AI 调用体验；真正的核心，是在此基础上提供可以直接使用、持续运行并完成任务的 Dola Agent。代理商的价值也不只是转售额度，而是把合适的 Agent、平台能力和服务方案带给具体用户。

## 代理商可以做什么

成为代理商后，你可以围绕自己的受众、行业资源或客户需求，建立一个可持续经营的 Agent 商店。

    代理合作方可以拥有独立的智能体店铺，用来组合、展示和销售适合自己客户群体的 Agent 与平台产品。
  </Card>
  <Card title="推广个人用户产品" icon="users">
    你可以把 Dola Agent 推荐给个人用户，帮助他们用于写作、学习、运营、资料整理和重复性工作处理。
  </Card>
  <Card title="服务企业客户" icon="building">

  </Card>
  <Card title="完成演示和交付" icon="handshake">
    平台产品额度可以用于自己体验、给客户演示、测试交付效果，也可以用于后续客户服务落地。
  </Card>
</CardGroup>

## 为什么不是单纯 Token 转售

Token 会越来越透明，用户也会越来越容易比较价格。只依赖 Token 差价的业务，长期利润空间会变薄。

Dola 更关注智能体应用本身：让 AI 不只是回答问题，而是能够通过 Agent 执行任务、完成工作并交付结果。你可以把 Dola Agent 理解为一个部署在云端、24 小时待命、可以通过微信或飞书入口被调用的 AI 执行助手。

## 早期代理政策

<Info>
  早期代理政策以 [dolaio.cn/join](https://dolaio.cn/join) 页面展示为准。当前页面仅用于说明代理合作的基本机制和申请流程。
</Info>

在 2026 年 7 月 15 日之前，Dolaio.cn 开放早期合作政策：

- 原代理费用为 **2000 元**
- 早期优惠价为 **1000 元**
- 同时赠送 **1000 元以上等值平台产品额度**

也就是说，早期代理并不是单独支付一笔代理费，而是用 1000 元购买 1000 元的平台产品额度，同时获得代理合作资格。

该额度可用于：

- 自己体验平台能力
- 使用 Dola Agent
- 给朋友或客户演示
- 转售给个人用户
- 转售给企业客户
- 用于企业服务落地

## 申请流程

<Steps>
  <Step title="打开代理申请页">
    访问 [dolaio.cn/join](https://dolaio.cn/join)，阅读平台介绍和代理合作政策。
  </Step>
  <Step title="填写申请信息">
    按页面提示填写个人信息、所在城市、职业角色、客户资源、AI 使用基础和合作意向。
  </Step>
  <Step title="提交合作申请">
    确认你理解 Dolaio.cn 是基于 Token 聚合能力之上的智能体平台，并提交申请。
  </Step>
  <Step title="等待审核">
    平台通常会在 **1-3 个工作日** 内完成审核。审核通过后，会分配平台体验入口、Dola Agent 账号、平台产品额度、Agent 商店权限和合作资料。
  </Step>
</Steps>
