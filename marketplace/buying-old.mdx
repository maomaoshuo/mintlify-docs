Dola 市场可以让你快速找到可购买的 AI 商品和服务。AI Agent 可以购买后连接使用；结果型 Agent 可以直接调用并获得输出；AI 硬件产品和个人服务会按商品说明完成交付。

## 浏览市场

访问 [dolaio.cn](https://dolaio.cn)，并从主菜单进入 **Marketplace**。你可以浏览完整目录，也可以通过搜索和筛选缩小范围。

- **Search** — 输入与你想自动化的任务或所需能力相关的关键词。
- **Categories** — 按主题或使用场景浏览，发现你可能没想到要搜索的 Agent。
- **Filters** — 按使用场景、底层模型类型、价格区间和创作者评分筛选，只查看真正符合需求的商品。

购买前，打开商品页查看：
- **Sample outputs** — Agent 实际产出的示例
- **Creator profile** — 创作者背景以及在平台上的记录
- **Buyer reviews** — 已购买用户的真实评分和反馈

## 购买商品

<Steps>
  <Step title="打开商品页">
    点击市场中的任意商品，查看完整详情页，包括描述、示例输出、价格选项和买家评价。
  </Step>
  <Step title="购买或试用">
    点击 **Buy Now** 直接进入结账。如果该商品提供试用期或演示，点击 **Try** 可以在正式购买前先体验。
  </Step>
  <Step title="完成结账">
    检查订单，并使用 Dola 支持的支付方式完成付款。购买会被安全处理并即时确认。
  </Step>
  <Step title="按商品类型开始使用">
    支付确认后，AI Agent 会出现在你的 **My Agents** 面板中；结果型 Agent 会提供调用入口；硬件产品和个人服务会展示后续交付说明。
  </Step>
  <Step title="连接 AI Agent">
    在 **My Agents** 中打开 Agent 并点击 **Connect** 激活它。这个操作会把 Agent 连接到你的 Dola 账号，并让它准备好开始工作。
  </Step>
  <Step title="开始使用 Agent">
    Agent 已经上线。你可以开始对话、分配任务，或把它接入你的工作流，无需额外设置。
  </Step>
</Steps>

## 购买之后

连接 Agent 后，你与它的关系不会止步于激活。Dola 市场的设计目标是让购买后的体验持续变好，而不仅是购买当下完成交付。

- **随时访问** — 从面板打开 **My Agents**，查看你拥有的所有 Agent，包括已激活、已暂停或待连接状态。
- **完全控制激活状态** — 你可以随时激活、暂停或调整 Agent 设置。暂停 Agent 不会取消购买，准备好后可以再次激活。
- **自动更新** — 如果创作者改进了 Agent，例如升级底层模型、优化系统提示词或添加新能力，你会自动获得更新版本，不需要重新购买。
- **联系创作者** — 如果对 Agent 有疑问，或需要使用建议，可以通过商品页直接联系创作者。

## 购买建议

<Tip>
  购买前查看 Agent 的示例输出和买家评分。示例输出能告诉你实际效果，其他买家的良好记录则是 Agent 能兑现描述的最佳信号。
</Tip>

<Note>
  部分 Agent 包含初始试用期，让你可以在正式购买前测试产品。商品页上如果有 **Try** 按钮，建议先使用它。
</Note>

<Note>
  退款政策会因商品而异，并由各创作者设置。购买前，请滚动到商品描述底部查看创作者的退款条款，确认自己的权益边界。
</Note>
