Dola 市场为你构建或能够提供的产品提供全球展示入口。商品审核通过后，平台上的买家都可以发现它；每一笔购买都会进入你的收益余额。下面是上架第一个产品并在 Dola 建立可持续创作者业务所需了解的内容。

## 你可以销售什么

<CardGroup cols={2}>
  <Card title="AI Agent" icon="robot">
    一个已经完整构建、可直接激活的 Agent，拥有明确目标、系统提示词和工具集。这是市场中的核心商品类型。
  </Card>
  <Card title="Skill" icon="puzzle-piece">
    一个独立能力，例如离散工具或集成，买家可以把它添加到已有 Agent 中。
  </Card>
  <Card title="Agent 配置" icon="sliders">
    现有 Agent 的预设配置，包括你选择的模型、指令和设置，并作为可复用产品打包。适合垂直工作流和行业场景。
  </Card>
  <Card title="个人服务" icon="handshake">
    咨询、引导式配置、自定义构建，或任何你提供的人工协助。服务让买家可以直接与你合作，而不是只购买数字产品。
  </Card>
</CardGroup>

## 上架流程

<Steps>
  <Step title="完成产品构建">
    创建商品前，确保你的 Agent、Skill 或配置已经完成，并能按预期工作。如果还在构建中，请查看 [Agent 创建指南](/agents/building) 获取逐步说明。
  </Step>
  <Step title="在 Creator Dashboard 中打开 My Listings">
    进入 **Creator Dashboard**，然后选择 **My Listings** → **New Listing** 开始上架流程。
  </Step>
  <Step title="选择产品类型">
    选择 **Agent**、**Skill**、**Configuration** 或 **Service**。你的选择会决定下一步出现哪些字段和设置。
  </Step>
  <Step title="填写商品详情">
    提供清晰标题、完整描述、截图或演示视频、相关标签和最合适的分类。信息越充分，合适买家越容易找到你的商品。
  </Step>
  <Step title="设置价格">
    选择定价模式：**一次性购买** 或 **订阅**（月付或年付）。如果产品适合两种方式，也可以在同一商品中提供两个选项。
  </Step>
  <Step title="设置收款方式">
    收益到账前，你需要在 **Account Settings** → **Payments** 中添加收款方式。Dola 支持多种提现选项，尽早配置可以确保从第一天起就能收款。
  </Step>
  <Step title="提交审核">
    商品信息确认无误后，提交审核。Dola 团队会检查商品质量、准确性以及是否符合市场规范。多数审核会在较短周期内完成。
  </Step>
  <Step title="正式上线">
    审核通过后，商品会立即出现在市场中。买家可以发现、购买并激活你的产品。
  </Step>
</Steps>

## 销售建议

<Tip>
  为商品添加清晰截图或简短演示视频。买家在付费前希望看到自己会得到什么。展示 Agent 实际运行效果，是提升转化率最有力的方式之一。
</Tip>

<Tip>
  写一段详细描述，包含真实使用场景和示例输出。具体性会建立信任。不要只说“一个有用的写作助手”，而要说明它处理哪类写作，以及输出长什么样。
</Tip>

<Tip>
  快速回复潜在买家和已有买家的问题。及时、有效的回应会提升卖家声誉，并直接影响评分；评分又会影响商品在搜索结果中的展示位置。
</Tip>

<Note>
  Dola 会从每笔销售中扣除平台费用。扣费后的净收入始终会显示在 **Earnings Dashboard** 中，因此你可以清楚知道每笔交易前后实际会收到多少。
</Note>

## 管理商品

Creator Dashboard 是你管理市场商品的控制中心。你可以在这里：

- **编辑或更新** 任意商品，包括修改描述、替换截图、调整价格或更新底层 Agent。已购买 Agent 的买家会自动收到你推送的更新。
- **暂停商品**，用于临时休息或进行较大调整。暂停后的商品不会对新买家可见，但已有客户的权益仍会保留。
- **查看单商品数据分析**，包括总销售额、平均评分、买家评价和历史收入，帮助你判断什么有效，并继续加倍投入。
