Dola 市场是一个经过组织的商品入口。AI 创作者可以在这里发布即开即用的 Agent 和数字产品，买家则可以无需任何设置或部署，直接发现、购买并激活它们。无论你想扩展 Dola 能为你做的事，还是已经构建了值得分享的产品，市场都是供需即时连接的地方。

## 市场中有什么

市场包含四类产品，每一类都旨在让买家快速获得价值，并让创作者从自己的工作中获得收入。

## 买家如何使用

浏览市场时，你可以在花钱之前查看每个商品的描述、示例输出、创作者资料和买家评价。找到适合需求的 Agent 或 Skill 后，购买并激活通常不到一分钟。

1. **浏览** — 按关键词、分类、使用场景或价格区间搜索。
2. **购买** — 使用支持的支付方式安全结账。
3. **连接** — 新 Agent 会立即出现在 **My Agents** 中。点击 **Connect** 激活后即可开始工作。

没有部署流程，没有配置文件，也没有等待期。连接完成的那一刻，Agent 就已经上线可用。

## 卖家如何使用

如果你已经构建了 Agent、组装了一套有价值的配置，或开发了其他人会需要的 Skill，就可以把它上架到市场并开始获得收入。你可以自行设置价格，选择一次性购买或订阅。Dola 会处理商品展示、结账和交付。商品审核通过后，平台上的所有买家都能看到它。

Creator Dashboard 会实时展示你每个商品的销售、评分和收入。你可以随时更新、暂停或移除商品，同时保留历史数据。

## 推荐返佣层

你不必是卖家，也可以在市场中获得收益。每个 Dola 用户，无论是买家还是创作者，都有一个唯一推荐链接。分享链接后，只要有人通过它完成合格购买，你就能获得佣金。即使你从未构建过 Agent，也可以通过受众或社群变现。
