Dola Agent 商店是一个经过组织的商品入口。创作者可以在这里发布可持续运行的 Agent、结果型 Agent、AI 硬件产品和个人服务，买家则可以直接发现、购买并使用它们。无论你想购买一个能立即上岗的 Agent，还是已经构建了值得分享的产品，Agent 商店都是供需即时连接的地方。

## Agent 商店中有什么

Agent 商店包含四类产品，每一类都旨在让买家快速获得价值，并让创作者从自己的工作中获得收入。

## 买家如何使用

浏览 Agent 商店时，你可以在花钱之前查看每个商品的描述、示例输出、创作者资料和买家评价。找到适合需求的商品后，按商品类型完成购买、调用、激活或预约服务。

1. **浏览** — 按关键词、分类、使用场景或价格区间搜索。
2. **购买** — 使用支持的支付方式安全结账。
3. **使用** — AI Agent 会进入你的 Agent 列表；结果型 Agent 可直接调用；硬件产品和个人服务会按商品说明完成交付。

对于 AI Agent，买家不需要处理部署流程或模型配置。连接完成的那一刻，Agent 就已经上线可用。

## 卖家如何使用

如果你已经构建了可长期运行的 Agent、结果型 Agent、AI 硬件产品或个人服务，就可以把它上架到 Agent 商店并开始获得收入。你可以自行设置价格，选择适合商品的购买方式。Dola 会处理商品展示、结账和交付流程。商品审核通过后，平台上的所有买家都能看到它。

Creator Dashboard 会实时展示你每个商品的销售、评分和收入。你可以随时更新、暂停或移除商品，同时保留历史数据。

## 推荐返佣层

你不必是卖家，也可以在 Agent 商店中获得收益。你可以在商品页生成个人商品码并分享给别人；只要买家通过个人商品码完成购买，系统就会按该商品的返佣规则分账。即使买家跳转到商店里的其他商品，代理关系也会按平台规则保留。
