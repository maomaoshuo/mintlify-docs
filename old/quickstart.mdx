---
title: "购买你的第一个 AI Agent"
sidebarTitle: "购买 AI Agent"
description: "从 Dola 精选市场选择 Agent，完成购买，并把 Agent 绑定到你的日常入口中使用。"
---

Dola 可以让你在几分钟内购买并启用一个已经配置好的 AI Agent。你只需要创建账号、选择商品、确认购买信息，然后进入 Agent 实例页完成入口绑定。

<Note>
  Dola 会员套餐会为 Agent 提供初始使用额度。购买后，你可以先真实使用 Agent，再根据自己的工作流决定是否长期续费或调整入口配置。
</Note>

<Steps>
  <Step title="创建 Dola 账号">
    前往 [dolaio.cn](https://dolaio.cn) 注册免费账号。你只需要一个邮箱即可开始。完成登录后，进入 Dola 精选市场浏览可购买的 Agent。
  </Step>

  <Step title="在 Dola 精选中选择 Agent">
    在 **DOLA精选** 页面，你可以按分类筛选 Agent，也可以通过搜索框输入名称或关键词。每张商品卡片都会展示 Agent 名称、简介、分类标签、创建者和价格。找到想购买的 Agent 后，点击商品卡片进入详情页。

    ![在 Dola 精选市场中选择 AI Agent](/images/quickstart/browse-agent-market.png)
  </Step>

  <Step title="打开商品页并点击购买">
    商品页会展示 Agent 封面、名称、价格和可用操作。确认这是你要购买的 Agent 后，点击 **购买**。如果你想把商品送给别人，也可以选择 **赠送**。

    ![在 AI Agent 商品页点击购买](/images/quickstart/open-agent-product.png)
  </Step>

  <Step title="确认购买详情">
    在购买详情弹窗中确认数量和服务器套餐。部分 Agent 需要运行在独立服务器上，你可以选择购买新服务器套餐，或安装在已有服务器中。确认总金额无误后，点击 **购买** 完成支付。

    ![确认购买数量、服务器套餐和总金额](/images/quickstart/confirm-purchase-details.png)
  </Step>

  <Step title="进入你的 Agent 实例页">
    购买成功后，Dola 会创建对应的 Agent 实例。你可以在实例页查看运行状态、到期时间、最近活动、免费用量、钱包余额和模型选择。页面中的 **入口链接** 区域会展示可连接的使用入口。

    ![查看购买后的 AI Agent 实例和入口链接](/images/quickstart/manage-agent-instance.png)
  </Step>

  <Step title="绑定微信入口">
    如果 Agent 支持微信入口，点击 **立即绑定** 打开微信管理设置。使用手机微信扫描二维码，并在手机端完成授权确认。绑定完成后，你就可以通过微信直接使用这个 Agent。

    ![扫码绑定微信入口](/images/quickstart/bind-wechat-entry.png)
  </Step>
</Steps>

<Tip>
  如果你只是想先体验，可以优先选择标记为 **免费** 的 Agent。购买后先完成一个入口绑定，再从实例页查看用量、模型和续费信息。
</Tip>
