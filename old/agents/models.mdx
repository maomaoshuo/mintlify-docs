为 Agent 选择 AI 模型，是创作者最关键的决策之一。它决定推理质量、回复语气、Agent 能处理的任务类型，以及买家从第一条消息开始获得的体验。Dola 让你可以从多个主流 AI 提供商的领先模型中自由选择，并且可以随时更换模型，不会打断买家的使用。

## Dola 上的模型自由度

Dola 不会把你锁定在单一 AI 提供商或单一模型层级中。创建或编辑 Agent 时，你可以从精选的高质量模型中选择驱动 Agent 的模型。这意味着你可以让模型匹配 Agent 的具体目标，而不是让所有 Agent 都使用同一个默认选择。

<Note>
  具体可用模型可能会随时间变化，因为模型提供商会发布新模型，也可能下线旧模型。请在 Agent 编辑器的 **Model** 下拉菜单中查看当前可用列表。
</Note>

## 如何选择模型

你可以在创建 Agent 时或发布后随时选择或更换模型：

<Steps>
  <Step title="打开 Agent 编辑器">
    从 [dolaio.cn](https://dolaio.cn) 的 Creator Dashboard 中，点击你想配置的 Agent；如果是从零开始，则点击 **New Agent**。
  </Step>
  <Step title="进入 Model 下拉菜单">
    在 Agent 编辑器中找到 **Model** 区域。点击下拉菜单查看完整可用模型列表。
  </Step>
  <Step title="选择模型">
    选择最符合 Agent 目标的模型。下拉菜单会展示每个模型的关键特征，帮助你判断。
  </Step>
  <Step title="保存变更">
    点击 **Save**。如果这是新 Agent，则点击 **Publish**。所选模型会立即对所有对话生效；如果你更新的是已发布 Agent，已有买家也会立即使用新模型。
  </Step>
</Steps>

## 按使用场景选择模型

不同模型有不同强项。你可以参考下面的类别，把模型与 Agent 的主要用途匹配起来。

<CardGroup cols={1}>
  <Card title="对话与推理型 Agent" icon="comments">
    如果 Agent 的核心工作是进行有上下文、有判断力的对话，例如回答问题、提供决策建议、解释复杂主题或处理客户支持，请选择推理和自然语言理解能力强的模型。这类模型通常能更好处理长上下文，并保持多轮对话连贯。

    **适合：** 助手、顾问、导师、客服 Agent、研究助手
  </Card>
  <Card title="图像生成型 Agent" icon="image">
    如果 Agent 需要创建、编辑或分析视觉内容，请选择具备视觉能力或图像生成能力的模型。有些模型既能理解用户上传的图片，也能根据提示词生成新图片。

    **适合：** 创意助手、设计工具、视觉内容生成器、图像编辑助手
  </Card>
  <Card title="代码与技术型 Agent" icon="code">
    如果 Agent 主要处理代码，例如编写、审查、调试或解释代码，请选择面向技术任务优化的模型。这类模型通常在大型代码库上训练，对编程语言、框架和软件工程概念有更精确的理解。

    **适合：** 编程助手、代码审查工具、调试工具、文档生成器、DevOps 助手
  </Card>
  <Card title="通用型 Agent" icon="sparkles">
    如果 Agent 需要处理多种任务，并没有单一主导场景，高质量通用模型通常能在不同领域之间取得较好平衡。这类模型可以同时胜任写作、分析、问答、总结和推理。

    **适合：** 通用效率助手、多领域研究工具、个人 AI 伙伴
  </Card>
</CardGroup>

## 会员套餐初始额度

Dola 会员套餐会为 Agent 提供初始使用额度。这意味着买家可以先与 Agent 互动，并使用你选择的模型，再根据自己的使用情况决定是否长期续费或调整套餐。机制如下：

- **初始额度：** 买家首次使用 Agent 时，会优先消耗会员套餐中的可用额度。
- **额度之后：** 初始额度用完后，后续使用会根据买家所在会员套餐或钱包余额继续计费。

这个设计是有意为之。移除尝试新 Agent 的财务风险后，买家更愿意深入体验你的产品，从而带来更好的体验、更强的评价和更高的长期留存。

## 发布后更新模型

你不会被原始模型选择锁定。你可以随时在 Creator Dashboard 的 **Agent settings** 中更换驱动 Agent 的模型。

<Note>
  更新已发布 Agent 的模型后，变更会立即生效。所有已有买家都会自动开始使用更新后的模型，你不需要通知他们，也不需要创建新商品。买家体验会保持连续，只是底层模型换成了新的版本。
</Note>

这种灵活性让你的 Agent 能随着更好的模型出现而保持竞争力。Dola 上的优秀创作者会定期更新 Agent，以利用新模型发布带来的能力提升，从而改善已有买家体验，并吸引市场中的新买家。
