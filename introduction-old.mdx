Dola 是属于每个人的 SaaS 化独立 Agent 商店。你可以像搭建个人 Wiki 一样，创建自己的 Agent 市场，发布产品、代理商品，并从 AI 市场中开始赚钱。

在 Dola，你不只是使用一个公共市场。你可以拥有自己的商店入口，把专业经验、工具能力、精选 Agent 和服务组织成可销售的商品。Dola 会负责模型接入、商品交付、支付结算和分润追踪，让你把重点放在选品、销售和持续经营上。

## Dola 适合谁

**创作者** 可以在 Dola 创建自己的 Agent 商店，发布自有 AI Agent、结果型 Agent、AI 硬件产品或个人服务。你负责想法、产品和交付，Dola 负责模型接入、支付和结算。如果你有值得产品化的知识、流程或使用场景，就可以把它做成能持续产生收入的商品。

**推广者** 可以创建自己的 Agent 商店，代理平台上的优质产品并获得分润。Dola 官方会持续提供大量优质 Agent 供你销售，你也可以从其他商店中挑选适合自己受众的优质商品，组合成自己的选品池。

**买家** 可以在市场中浏览并购买即开即用的 Agent。完成购买后，Agent 会立即激活并可用。你不需要部署服务，不需要管理 API Key，也不需要处理复杂集成。打开后就能开始工作。

## Dola 有什么不同

### 模型选择自由
你可以从 Dola 精选的高质量模型中选择最适合 Agent 的模型。Dola 会员套餐会为 Agent 提供初始使用额度，让买家能先体验再决定是否长期使用，从而降低首次成交门槛。

### 不只销售 Agent
Dola 市场不只支持长期运行的 AI Agent。创作者也可以出售结果型 Agent、AI 硬件产品和个人服务。只要能为别人创造明确价值，就可以在 Dola 上产品化。

### 内置推荐返佣
每个用户都有专属推荐链接。分享链接后，只要有人通过你的链接完成购买，你就能获得佣金。推荐系统直接内置在平台中，不需要第三方工具，也不需要手动追踪。

### 端到端商业化
Dola 覆盖完整生命周期：创建 Agent，上架市场，交付给买家，再完成创作者收益结算。你不需要把创建工具、销售渠道和收款系统拼在一起。
