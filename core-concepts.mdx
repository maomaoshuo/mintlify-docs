Dola 围绕一组相互连接的概念构建。理解 Agent 是什么、Skill 如何扩展能力、Agent 商店如何连接创作者和买家，以及支付和推荐返佣如何流转后，你就能更自然地使用整个平台。

---

## Agent

Agent 是一个可配置的 AI 数字工作者。创作者定义 Agent 的名称、角色和指令，然后选择一个 AI 模型作为能力底座。买家从 Agent 商店购买 Agent 后即可立即激活，无需额外设置。从那一刻起，Agent 就可以接收任务、回答问题或按照创作者设计的方式运行工作流。

你可以把 Agent 理解成一个购买后马上能上岗的专家。

---

## Skill

Skill 是一种模块化能力，用来扩展 Agent 能做的事。Skill 可以接入 Agent，解锁特定动作，例如搜索网页、生成图片、调用外部 API 或处理文档。创作者可以在配置 Agent 时添加 Skill，并把这些能力组合进完整 Agent 或结果型 Agent 中交付给买家。

---

## Model

每个 Agent 都运行在一个底层 AI 模型之上。模型负责理解语言、推理问题并生成回复。Dola 支持一组精选的高质量模型，创作者可以在创建 Agent 时选择最适合目标场景的模型。

不同模型有不同强项。有的擅长结构化推理，有的擅长创意写作或代码生成。选择合适的模型，是创作者塑造 Agent 质量和性格的重要环节。

<Note>
  Dola 会员套餐会为 Agent 提供初始使用额度。买家购买后可以立即开始体验 Agent；创作者也能让新用户先看到真实效果，再决定是否长期使用。
</Note>

---

## Marketplace

Marketplace 是 Dola 的 Agent 商店，也是创作者上架产品、买家发现并购买产品的地方。商品可以是 24 小时独立运行的 AI Agent、直接调用即可获得结果的结果型 Agent、AI 硬件产品，也可以是创作者提供的个人服务。

Agent 商店会按分类和使用场景组织内容，方便买家快速找到所需产品。每个商品页都会展示价格、交付方式、能力说明，以及创作者撰写的介绍。

---

## Creator

Creator 指在 Dola Agent 商店创建并发布产品的用户。创作者设计 Agent，编写角色和指令，选择模型，组合 Skill，并设置价格。商品上架后，Dola 会负责交付和支付。创作者可以专注于打磨产品，而不是管理交易流程。

创作者可以从每一笔销售中获得收入，并通过内置推荐系统放大收益。

---

## Buyer

Buyer 指在 Dola Agent 商店购买 Agent、结果型 Agent、AI 硬件产品或个人服务的用户。完成购买后，平台会按商品类型完成激活、调用或交付。除非买家主动选择进一步自定义，否则不需要处理复杂技术步骤。

买家可以在账号中拥有多个 Agent，每个 Agent 服务于不同目标或使用场景。

---

## Referral & Commission

每个 Dola 用户，无论是创作者还是买家，都拥有一个专属推荐链接。当有人通过你的链接完成购买，你就能从该订单中获得佣金。佣金系统直接内置在平台中，无需额外安装或配置。

推荐返佣是 Dola 的重要增长机制。积极分享推荐链接的创作者，不只能获得商品销售收入，还能获得额外佣金；买家也可以通过推荐自己正在使用的 Agent 获得收益。

---

## Settlement

Settlement 指 Dola 处理每笔交易财务流转的方式。买家完成购买后，Dola 会处理支付，并把创作者收益排入结算流程。创作者通过平台内置结算系统收款，不需要单独接入支付处理方，也不需要手动管理发票。

结算会在后台自动运行，因此创作者可以专注于构建产品，买家可以专注于完成工作。

---

## 它们如何串起来

<Tip>
  Dola 被设计成一个完整闭环：Creator 创建 Agent 或准备其他可售商品，并上架到 Marketplace。Buyer 找到它，完成购买、调用或激活。Dola 处理支付并自动结算 Creator 收益。Referral 链接在传播过程中带来佣金。这个页面里的每个概念都服务于同一个闭环，并且都在一个平台内完成。
</Tip>
